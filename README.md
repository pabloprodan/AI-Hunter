# AI Hunter

### Work Smarter. Win Faster.

**AI Hunter is an AI-powered workspace that helps freelancers discover opportunities, organize their pipeline, and grow their business.**

---

## Mission

Build the operating system for the modern freelancer — one that doesn't just track work, but actively helps you find, win, and deliver it.

## Structure

```
ai-hunter/
├── apps/
│   ├── web/          # Next.js frontend
│   └── api/          # API server
├── packages/
│   ├── ui/           # Design system components
│   ├── core/         # Business logic
│   ├── types/        # Shared TypeScript types
│   ├── config/       # Shared configuration
│   ├── ai/           # AI orchestration layer
│   └── utils/        # Shared utilities
├── docs/
│   ├── bible/        # Product bible
│   ├── architecture/ # Architecture guides
│   ├── decisions/    # ADRs
│   └── roadmap/      # Product roadmap
├── scripts/          # Automation scripts
└── assets/           # Static assets
```

## Stack

- **Frontend:** Next.js, React, TypeScript, Tailwind CSS
- **Backend:** Node.js, TypeScript
- **AI:** Multi-model orchestration (via Open Cowork ecosystem)
- **Infrastructure:** Turborepo, npm workspaces
