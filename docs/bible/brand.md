# AI Hunter Brand Guidelines

## Identity

AI Hunter is an AI-powered workspace that helps freelancers discover opportunities, organize their pipeline, and grow their business.

**Tagline:** Work Smarter. Win Faster.

**Archetype:** The Hunter — proactive, precise, empowering.

## Logo

- **Symbol:** A crosshair/target icon representing precision and opportunity
- **Wordmark:** "AI HUNTER" in bold sans-serif, all caps
- **Preferred lockup:** Symbol to the left of wordmark

## Colors

| Token | Value | Usage |
|-------|-------|-------|
| Primary 600 | `#4f46e5` | Buttons, links, primary actions |
| Primary 500 | `#6366f1` | Hover states |
| Success 500 | `#10b981` | Won deals, active, positive metrics |
| Warning 500 | `#f59e0b` | Pending, in-progress, attention |
| Danger 500 | `#ef4444` | Lost deals, errors, alerts |
| Neutral 900 | `#0f172a` | Headings, primary text |
| Neutral 500 | `#64748b` | Secondary text, icons |
| Neutral 100 | `#f1f5f9` | Backgrounds, cards |

## Typography

- **Headings:** Inter, Bold/Semibold
- **Body:** Inter, Normal/Medium
- **Monospace:** JetBrains Mono (code, data displays)

## Tone of Voice

- Direct and confident — no fluff
- Empathetic but not emotional
- Speaks to freelancers as professionals, not beginners
- Uses active voice: "AI Hunter finds opportunities" not "Opportunities are found"

## Spacing

Base unit: 4px. All spacing follows a 4px grid: 4, 8, 12, 16, 20, 24, 32, 40, 48, 64.

## Component Principles

1. **Clarity first** — every component should be self-explanatory
2. **Data-forward** — show metrics, numbers, scores prominently
3. **Action-oriented** — CTAs should be obvious and contextual
4. **AI-native** — AI-generated content is labeled, not hidden

## States

Components communicate state clearly:

- **Loading:** Skeleton screens, not spinners (unless action-based)
- **Empty:** Illustration + message + CTA
- **Error:** Actionable message with retry
- **Success:** Confirmation + next step
